# PostgreSQL Practice

This repository contains my PostgreSQL practice queries and examples.

## Topics Covered

1. Create Table
2. Insert Data
3. Select Statements
4. WHERE Clause
5. UPDATE Statement
6. DELETE Statement
7. ORDER BY
8. LIMIT
9. Aggregate Functions
   - COUNT()
   - SUM()
   - AVG()
   - MAX()
   - MIN()
10. GROUP BY
11. HAVING Clause
12. Joins
    - INNER JOIN
    - LEFT JOIN
    - RIGHT JOIN
    - FULL OUTER JOIN
    - CROSS JOIN
    - SELF JOIN
13. Subqueries
    - Single Row Subqueries
    - Aggregate Subqueries

## Learning Outcomes

- Created and managed database tables.
- Performed CRUD operations (Create, Read, Update, Delete).
- Filtered records using WHERE conditions.
- Sorted and limited query results.
- Used aggregate functions for data analysis.
- Grouped data and applied conditions using HAVING.
- Combined multiple tables using different types of JOINs.
- Retrieved data using nested queries (Subqueries)
